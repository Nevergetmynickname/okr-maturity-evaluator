# OKR Maturity Evaluator Skill

A mature OKR should not only ask, “Can we achieve the target?” It should also ask, “What might we lose if we achieve this target too aggressively?”. 

Strong OKRs balance ambition with sustainability. They prevent teams from over-optimizing one dimension, such as speed, cost, sales, or delivery volume, at the expense of quality, trust, resilience, employee capacity, or long-term growth. 

This skill is for evaluating the maturity of an organization's OKRs and identify gaps based on above priciple. 

Use this skill when the user provides organizational background, role and responsibility, strategic targets, and OKRs, and asks for OKR diagnosis, OKR maturity assessment, OKR alignment review, gap analysis, or recommendations for improving OKR quality and execution.

## What it does

This skill evaluates OKRs across:

- Strategic alignment
- Objective quality
- KR measurability
- Execution grip--（whether the OKR has enough process-level handles to make execution manageable）
- Ownership and accountability
- Review and governance
- Balance between goal achievement and sustainable growth

## Expected input

The skill works best when the user provides:

- Organization background
- Role and responsibility
- Strategic target
- Current OKRs
- Related tasks or initiatives
